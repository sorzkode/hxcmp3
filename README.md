# [hxcmp3.com Music Archive ↗️](https://hxcmp3.sorzkode.com/)
Another early 2000s preservation project by sorzkode.  

## Contents
The repository hosts all music files which can be downloaded in bulk via [Releases](https://github.com/sorzkode/hxcmp3/releases/tag/vMOSH)  
- [chunk1.zip](https://github.com/sorzkode/hxcmp3/releases/download/vMOSH/chunk1.zip) -- Contains artists up to the letter N.
- [chunk2.zip](https://github.com/sorzkode/hxcmp3/releases/download/vMOSH/chunk1.zip) -- Contains M onward.

> Recognize a band or song in the unknown category? Open a [pull request](https://github.com/sorzkode/hxcmp3/pulls) and use the PR template.

## Methodology 
I used Python to leverage archive.org's API to find all music files it archived from hxcmp3.com. From there, I used a variety of scripts, tools, and manual effort to tag and catalog the files.  

## Legality
To my knowledge, all the bands represented here have been defunct for many many years. I assume that any of the bands represented would be happy to have their musical legacy preserved. However, if you have legal ownership of any of the files in this collection and want me to remove it, open [an issue](https://github.com/sorzkode/hxcmp3/issues) and I will review.

## History Lesson
Don't know wtf hxcmp3.com is or what it was like on the internet in the early 2000s?

hxcmp3.com was a community-driven music hosting platform that served the hardcore, punk, and metal scenes. Users of the site could:
- Upload audio files.  
- Create band profiles.  
- Stream tracks.
- Download songs for free.  

This type of site was commonplace back then (IE mp3.com, MySpace Music, PureVolume etc.)
